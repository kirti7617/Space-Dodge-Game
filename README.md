# Space-Dodge-Game
AI-powered image detection using React, FastAPI, PyTorch CNN, EXIF metadata, and Error Level Analysis (ELA)
## ✨ Features

* 🤖 **Hybrid AI Detection Engine** – Combines a custom PyTorch CNN with image forensic techniques for accurate AI image detection.
* 🖼️ **AI vs Real Image Classification** – Classifies uploaded images as AI-generated or authentic with a confidence score.
* 🔍 **Error Level Analysis (ELA)** – Analyzes compression artifacts to detect signs of AI-generated or manipulated images.
* 📊 **Metadata & EXIF Analysis** – Inspects image metadata and provenance information to support predictions.
* 🎨 **Modern Glassmorphism UI** – Clean, responsive interface with drag-and-drop image upload and Vanta.js animated background.
* ⚡ **FastAPI Backend & Local Processing** – Processes images efficiently on your local machine with a lightweight FastAPI server.
* 🧠 **Custom Trainable CNN Model** – Includes a PyTorch-based CNN that can be trained locally using the CIFAKE dataset for reliable performance.
